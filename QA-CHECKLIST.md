# QA Checklist

Pengujian dilakukan pada viewport mobile 390 × 844 piksel menggunakan browser Chromium headless.

## Pengujian yang lulus

- [x] Splash → onboarding → beranda.
- [x] Pilihan pesan mencurigakan.
- [x] Tiga langkah pemeriksaan risiko.
- [x] Hasil pemeriksaan dan rekomendasi tindakan.
- [x] Jalur “data sudah dikirim”.
- [x] Langkah pengamanan.
- [x] Pemilihan pendamping.
- [x] Ringkasan pendamping.
- [x] Modal “Tunjukkan kepada pendamping”.
- [x] Klinik Privasi dan rekomendasi dinamis.
- [x] Laboratorium Verifikasi.
- [x] Pengaturan perbesar teks.
- [x] Pengaturan kurangi animasi.
- [x] Tidak ditemukan error JavaScript pada alur yang diuji.
- [x] Server lokal berbasis Node.js menyajikan HTML, CSS, JavaScript, SVG, dan web manifest.
- [x] File JSON valid.
- [x] Sintaks JavaScript valid.

## Pemeriksaan sebelum pitching

- [ ] Buka web pada perangkat yang benar-benar digunakan saat lomba.
- [ ] Jalankan seluruh alur demo sekali tanpa berbicara.
- [ ] Pastikan zoom browser 100%.
- [ ] Pastikan koneksi internet stabil atau web sudah pernah dimuat agar cache offline tersedia.
- [ ] Simpan video rekaman demo sebagai cadangan.
- [ ] Jangan memasukkan data pribadi sungguhan selama demonstrasi.
